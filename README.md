# SpongeBob Themed Discord Bot

## Project Description
This SpongeBob-themed Discord bot is designed to bring joy and fun to your Discord server by providing various SpongeBob-related commands, games, and interactions. The bot utilizes SpongeBob quotes, images, and other multimedia elements to engage users and lighten the mood with humor from the beloved animated series.

## Features
- **SpongeBob Quotes**: Get random quotes from your favorite SpongeBob characters.
- **Games**: Enjoy interactive games based on SpongeBob episodes and themes.
- **Custom Commands**: Create and manage custom commands related to SpongeBob.
- **Image Gallery**: Share images from the show, including GIFs and fan art.

## Installation Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/jadegold55/first-discord-bot.git
   cd first-discord-bot
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Create a .env file** and add your Discord bot token:
   ```bash
   DISCORD_TOKEN=your-bot-token
   ```
4. **Run the bot**:
   ```bash
   node index.js
   ```

## Usage Guide
- To start using the bot, invite it to your Discord server and ensure it has the necessary permissions.
- Use commands starting with `!sponge` to access SpongeBob features. For example, `!sponge quote` to get a random SpongeBob quote.
- Check the help command with `!sponge help` for a list of available commands and their usage.
